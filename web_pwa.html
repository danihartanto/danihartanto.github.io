Web Tracer Study Alumni

Project Information

Project Name

  • Web Tracer Study Alumni

Project Description

  • Website ini merupakan aplikasi yang dirancang untuk membantu kampus dan alumni dalam melakukan pelacakan (tracing) terhadap lulusan/alumni mereka. Website ini memungkinkan alumni untuk memperbarui data pribadi, riwayat pendidikan, aktivitas karier, hingga pengalaman studi lanjutan mereka. Informasi yang dikumpulkan berguna untuk mengevaluasi kualitas pendidikan, mendukung akreditasi, serta membangun jejaring antara alumni dan institusi.

Project Frameworks

  • NodeJS v22
  • ExpressJs v4
  • ReactJs v18

Project Duration

  • 4 Months

Project Features

  • Input Data Diri Alumni
  • Manajemen Data Alumni
  • Manajemen Lowongan Pekerjaan
  • Laporan Tracing Alumni
  • Kuisioner Alumni & Instansi
